# 🚀 Team Productivity Hub

A production-grade team productivity platform with real-time chat, task management, habit tracking, notes, reminders, and analytics.

## Tech Stack

- **Frontend**: React 18 + TypeScript + Vite + Tailwind CSS + shadcn/ui
- **Backend**: Node.js + Express.js + TypeScript
- **Database**: MongoDB + Mongoose
- **Realtime**: Socket.io
- **Auth**: JWT + bcrypt

## Getting Started

### Prerequisites
- Node.js 18+
- MongoDB (local or Atlas)

### Server
```bash
cd server
cp .env.example .env  # Configure your environment
npm install
npm run dev
```

### Client
```bash
cd client
npm install
npm run dev
```
